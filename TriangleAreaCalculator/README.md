# Калькулятор площади треугольника

Веб-приложение для вычисления площади треугольника по основанию и высоте.

## Формула
S = (a × h) ÷ 2

## Использование
1. Введите длину основания
2. Введите высоту треугольника
3. Нажмите "Вычислить площадь"

## Технологии
- HTML5
- CSS3
- JavaScript
